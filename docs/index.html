The Context & Mission

About The EcoTrace Mission

Greenhouse calculators shouldn't be about endless spreadsheets or boring cold reports. EcoTrace AI reimagines sustainability by mapping cold, physical metrics into actionable visual models. With an integrated local inference state and Gemini AI client, it enables intelligent forecasting.

🎯

Primary Purpose

Establish custom user baselines using home energy footprints, commute parameters, and weekly grocery profiles. These values formulate precise calculation blocks instantly.

🧠

How AI Empowers Your Reduction

The integrated **Google Gemini 3.5 API Client** works securely against dynamic user states. The engine understands adjustments instantly, sending tailored warnings, supportive encouragement, and scenario advice straight to your tracker dashboard.

Quick Deployment

Setup & Environment Guide

Easily clone the open-source repository and customize variables locally.

1 Clone The Codebase

git clone https://github.com/errollajayendra/ecotrace-ai.git

2 Configure Local Private Properties

Securely manage properties in `.env` sheets locally. The variables automatically compile to protect keys from committing to public history.

Create root `.env` values:

GEMINI_API_KEY=AIzaSy...yourLiveKeyHere...

3 Build Standalone APK File

To compile, compile easily by invoking native gradle setups on terminal window:

gradle :app:assembleDebug
Saves output in app/build/outputs/apk/debug/app-debug.apk
