About the EcoTrace Project

EcoTrace AI bridges the gap between passive carbon calculation and active emission reduction. By translating cold statistical indicators into real-world equivalents (e.g., "smartphones charged"), users build a contextual relationship with their everyday choices.

🎯

Primary Purpose

EcoTrace AI provides a highly personal Carbon Baseline calculator based on home energy consumption values, commute behaviors, and general food and dietary habits. These values compile mathematically into dynamic totals, helping you easily identify areas of peak carbon emissions.

🧠

How AI Empowers Your Carbon Reduction

The embedded **Google Gemini Client** evaluates real-time delta logs against your custom dashboard data. Rather than displaying generalized advice, the AI analyzes your specific, granular daily changes to provide smart insights, interactive simulated scenarios, and actionable sustainability tips.

Install & Set Up Locally

Designed to be completely cloneable. Follow these standard instructions to run or build.

1. Clone the Source Repository Shell CLI

git clone https://github.com/errollajayendra/ecotrace-ai.git

2. Build the Duplicate .env File Environments

Duplicate `.env.example` in the root folder, and name it exactly `.env` (it is automatically ignored by Git):

GEMINI_API_KEY=YOUR_ACTUAL_API_KEY_HERE

3. Load inside Android Studio

Start Android Studio and select **"Open an Existing Project"**. Gradle will compile all required libraries. Click the **Run** top button (or press `Shift + F10`) to launch directly into an emulator!
